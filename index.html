How to Use This App

  1. Click on the upload area to select an image file
  2. After the image preview appears, click the "Process Image" button
  3. Wait for the YOLO algorithm to process your image
  4. View the detected objects in the results section
  5. For best results, use images with clear objects and good lighting

Deployment Notes

This frontend is designed to connect to a YOLO backend API. To fully deploy your app:

  1. Host your YOLO model on a cloud service (Heroku, AWS, Google Cloud, etc.)
  2. Update the API endpoint in the JavaScript code
  3. Enable CORS on your backend to allow requests from GitHub Pages
